# UC Reporter — What's Coming Next

A living view of where UC Reporter is headed. This is a beta, so priorities move
with your feedback — dates aren't promises, but direction is real.
Last updated: **2026-07-07**.

## ✅ In the current beta

- **Region → Location → Cluster** estate modelling and a report that reads
  cleanly for multi-site customers.
- **Multi-cluster collection** across CUCM, Unity Connection, IM&P, UCCX,
  Expressway, and CUBE/ISR — all local and offline.
- **Resource & performance metrics** (CPU / memory / storage per host) and a
  redesigned maintenance report with an at-a-glance overview dashboard.
- **Lifecycle & security insight** — end-of-life status and known-issue
  awareness from a built-in knowledge base.
- **Polished PDF export** with an executive summary, hierarchy, inventory,
  resource, and health sections.

## 🔨 In progress / next up

- **Deeper database-replication health** for Unity Connection and UCCX.
- **SIP-trunk and dial-plan reporting.**
- **Expanded capacity collection** from CUCM (AXL) for sizing insight.
- **Signed installer** — production releases will be code-signed, so the
  "unknown publisher" prompt goes away.
- **More report polish** driven directly by beta feedback.

## 🔭 Exploring / later

- **Online license activation & renewal** (keeping an offline mode for
  air-gapped environments).
- **Seat management** for teams.
- **Additional export formats and report templates.**
- **Optional, opt-in** crash/usage reporting to help us prioritise fixes.

---

_This roadmap is a good-faith snapshot and may change. It doesn't create any
commitment or obligation. Features ship when they're ready._
